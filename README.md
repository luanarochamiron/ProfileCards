# Responsive Profile Cards Section

Uma seção de cartões de perfil (Profile Cards) moderna, alinhada e totalmente responsiva desenvolvida com HTML5 e CSS3. O projeto simula uma página de equipe ou área de depoimentos e avaliações, organizando múltiplos cartões de usuários em uma grade fluida que se adapta a diferentes tamanhos de tela.

## Tecnologias Utilizadas

* HTML5: Estruturação semântica do contêiner geral e dos cards individuais de perfil.
* CSS3: Estilização completa, uso de bordas arredondadas, sombras discretas e gerenciamento de cores de marca (roxo e branco).
* Flexbox / CSS Grid: Alinhamento responsivo dos cartões em fileira e organização interna dos elementos de cada perfil.

## Funcionalidades e Técnicas Aplicadas

* Layout em Grade Responsiva: Distribuição paralela de múltiplos cards que se reorganizam de forma fluida dependendo da resolução da viewport.
* Molduras de Imagem Circulares: Fotos de perfil estilizadas em formato redondo com contornos roxos elegantes integrados à paleta de cores principal.
* Sistema de Classificação por Estrelas (Rating): Inclusão de blocos de avaliação visual com estrelas para indicar o nível de pontuação ou engajamento de cada perfil.
* Botões de Ação Duplos: Cada cartão possui dois botões inferiores com estilos complementares ("Read More" com bordas e "Subscribe" com preenchimento roxo sólido) que oferecem ótimo contraste.
* Organização Visual Confortável: Uso equilibrado de margens e preenchimentos para criar um layout limpo e de fácil leitura sobre um fundo roxo de destaque.
